# Shuobj
Shuobj (SHU .OBJ) is a single header, cross platform .obj file parser library.

It uses the [SHU](https://github.com/omerfuyar/shu) system. By defining `SHU`, you can tell the library where to find `shu.h`.

Goal is to easily parse .obj files and their utilities. It outputs all the information in a .obj file and all of it's helpers like .mtl, textures etc.
